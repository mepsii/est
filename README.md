----

<h2>implemented features:</h2>

- day and night cycle
- multiple mobs
- 3 projectile based guns, 2 tools
- enemies
- cell based interior system
- random map generation
- basic crafting
- resource gathering
- inventory system
- 2 biomes
- food and stamina bars
